WELCOME TO GUILDMANAGER v1.1.0

First things first ! You should read our board first : http://www.guildmanager.eu
There you will have the last information available on the project.

Here is a short "how to" for installation procedure :

1. In your database management software (ex. phpMyAdmin), import file GuildManager_SetUp.sql

2. Edit file "resources/config.php" : replace the title, the language, the GW2 server ID and most important the authorization for groups (you get the group number from phpBB's ACP)

3. With your FTP software (ex. FileZilla), copy all the files in the phpBB's directory, next to adm, cache, etc... (You can suppress the .sql file first.)

4. Go to "your-board-url/GuildManager", enjoy !

BIENVENUE DANS GUILDMANAGER v1.1.0

Tout d'abord, vous devriez lire le forum : http://www.guildmanager.eu
Il contient les dernières informations sur le projet.

Voila une rapide procédure d'installation :

1. Dans votre logiciel de gestion de base de données (ex. phpMyAdmin), importer le fichier GuildManager_SetUp.sql

2. Editez le fichier "resources/config.php" en remplaçant le titre et surtout les numéros des groupes autorisés (ce numéro peut s'obtenir via le PCA de phpBB) 

3. Dans votre logiciel ftp (ex. FileZilla), copiez l'ensemble des documents dans le dossier du forum, au même niveau que adm, cache, etc... (vous pouvez supprimer le fichier .sql avant la copie)

4. Allez à l'adresse "adresse-de-votre-forum/GuildManager", enjoy !
